# Real-Time Multimodal Communication Platform for Speech and Hearing Impaired Individuals

A production-ready full-stack platform enabling real-time communication through speech, text, and sign language.

## Monorepo Structure

- frontend: React + Vite + Tailwind + PWA
- backend: Django + DRF + Channels + JWT + PostgreSQL + Redis
- ai-services: FastAPI AI microservices for STT, TTS, and sign detection
- docker: Docker compose and service Dockerfiles
- docs: deployment, API, and architecture guides
